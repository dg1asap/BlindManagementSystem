Detailed Description

Elements values convention: XXXYYYYYb.

  • YYYYY : Flag position in the register
  • XXX : Register index
    • 001: CR register
    • 010: CSR register
    • 011: CRRCR register (*) (*) Applicable only for STM32L052xx, STM32L053xx, (...), STM32L073xx & STM32L082xx
